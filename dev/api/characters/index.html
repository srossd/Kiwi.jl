Characters API

Computing Characters

Kiwi.character — Method
character(rep::Irrep)

Compute the complete character of an irreducible representation. Returns a Character object containing all weights with their multiplicities.

Uses Freudenthal's multiplicity formula, which recursively computes the multiplicity of each weight μ from the multiplicities of weights μ + kα for positive roots α.

Algorithm

Starting from the highest weight λ (with multiplicity 1), we work our way down by subtracting positive roots. For each new weight μ, Freudenthal's formula gives:

2(λ+ρ, μ) m(μ) = Σ_{α>0} Σ_{k≥1} 2(μ+kα, α) m(μ+kα)

where the sum is over all positive roots α and all k such that μ+kα is a weight.

Returns

A Character object with a dictionary mapping weights to multiplicities.

References

  • Humphreys, "Introduction to Lie Algebras and Representation Theory" (1972), §22.3
  • Fulton & Harris, "Representation Theory" (1991), §15.2
