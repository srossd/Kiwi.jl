Types

Kiwi.LieAlgebra — Type
LieAlgebra

Represents a simple Lie algebra by its type and rank. Supported types: A, B, C, D (classical) and E, F, G (exceptional).

source
Kiwi.Irrep — Type
Irrep

Represents an irreducible representation of a Lie algebra by its highest weight. The highest weight is specified by Dynkin labels.

source
Kiwi.Rep — Type
Rep

Represents a (possibly reducible) representation as a linear combination of irreps. Stored as a dictionary mapping irreps to their multiplicities.

source
Kiwi.Weight — Type

Weight vector in the weight space of a Lie algebra. Weights are represented as linear combinations of fundamental weights: w = Σ ci ωi.

source
Kiwi.AbstractCharacter — Type
AbstractCharacter

Abstract base type for character representations. Subtypes must implement Base.getindex(char, weight) to return multiplicities.

source
Kiwi.Character — Type
Character <: AbstractCharacter

Represents a fully computed character with all weights and multiplicities precomputed.

source
Kiwi.LazyCharacter — Type
LazyCharacter <: AbstractCharacter

Represents a lazily-evaluated character that computes weight multiplicities on demand. Only computes multiplicities for weights that are explicitly queried.

source
Kiwi.WeightMultiplicity — Type
WeightMultiplicity

A weight with its multiplicity in a representation.

source
